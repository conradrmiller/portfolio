Crypto Dashboard

A dashboard with news and exchange rates

I made the Crypto Dashboard to be a spot where I can easily see the exchange rates between different crypto currencies and see the latest activity in crypto news. I wanted to give the page a bit of cyberpunk feeling. I just finished reading Snow Crash by Neal Stephenson so I had cyberpunk on my mind and Crypto being a virtual currency I felt like it fit. I also wanted a place to practice with css and web design. My hope is that people find the page useful, engaging, and beautiful.

Crypto Dashboard

The Crypto Dashboard allows the user to view exchange rates between crypto currencies. It also shows a list of the top 10 recent stories on the r/crypto subreddit. The project is built in React using CSS for styling and deployed on GitHub pages. The project utilizes open apis from Coinbase for calculating the exchange rate. The news comes from reddit and is the top 10 stories on the r/crypto subreddit. The news is gathered by an api call to the reddit api.
